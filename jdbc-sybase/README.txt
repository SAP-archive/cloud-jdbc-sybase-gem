= jdbc-sybase

* http://jruby-extras.rubyforge.org/activerecord-jdbc-adapter/

== DESCRIPTION:

This is a Sybase ASE JDBC driver gem for JRuby.

Use

   require 'jdbc/sybase'

to make the driver accessible to JDBC and ActiveRecord code running in JRuby.

=============================
Building the gem:

$> gem build jdbc-sybase.gemspec


=============================
Installing the gem: 

$> gem install jdbc-sybase
